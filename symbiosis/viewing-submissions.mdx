## Accessing the Procurement

This section covers how to access and navigate the Abatable platform for managing the Symbiosis RFP.

---

### URL

`https://app.abatable.com/`

---

### Navigation

#### Accessing the Procurement

1. Click **'My procurements'** in navigation
2. Select the **Symbiosis RFP** from the list
3. You'll see the procurement homepage

#### Procurement Homepage

The procurement homepage shows:
- RFP details
- Supplier invitations
- Submitted proposals - overview
- Constructed portfolios
- Actions to take

---

## Viewing Submissions

This section covers how to view and navigate supplier submissions in the platform, including the proposals table, filters, dashboard views, and proposal detail pages.

---

### Proposals Dashboard

#### Accessing Proposals

1. Once you have navigated to the **Symbiosis RFP**
2. Click **'Proposals'** or **'View Submissions'**
3. You'll see the proposals overview

#### Dashboard Overview

The dashboard displays:

| Component | Description |
|-----------|-------------|
| **Summary Numbers** | Total proposals, by stage, by status |
| **Pipeline Chart** | Visual breakdown by procurement stage |
| **Overview Charts** | Projects by country, type, developer type, etc. |
| **Recent Activity** | Latest submissions and updates |

#### Key Metrics

- Total submissions received
- Proposals per stage
- Pass/fail rates at eligibility check
- Average scorecard scores

---

### Proposals Table

#### Table Columns

The proposals table displays key information for quick comparison:

| Column | Description |
|--------|-------------|
| **Project Name** | Supplier's project |
| **Supplier Name** | Submitting organisation |
| **Project Type** | Reforestation, Agroforestry, Mangrove |
| **Country** | Project location |
| **Methodology** | Methodology and version |
| **Stage** | Current procurement stage |
| **Status** | Active, Rejected, Under Review, etc. |
| **Scorecard** | Category scores (Delivery, Methodology, Quality, Commercial) |
| **Appeal Indicator** | Shows if proposal has active appeal |
| **Price Offered** | Price per tonne (range) |
| **Volume Offered** | Volume range |
| **Submitted** | Submission date |
| **Latest Note** | Most recent note on the proposal |

#### Table Features

| Feature | Description |
|---------|-------------|
| **Sortable Columns** | Click column headers to sort; multi-level sorting supported |
| **Column Customisation** | Show/hide columns; reorder columns; saved per user |
| **Bulk Selection** | Select multiple proposals for mass actions |
| **New Indicators** | Unread/new proposals are highlighted |

#### Column Customisation

1. Click the **column settings** icon
2. Check/uncheck columns to show/hide
3. Drag columns to reorder
4. Settings are saved for your user account

---

### Filtering and Search

#### Search

Search proposals by:
- Project name
- Project ID
- Supplier name
- Proponent name

#### Filters

| Filter | Options |
|--------|---------|
| **Procurement Stage** | Select one or more stages |
| **Proposal Status** | Active, Rejected, Under Review, etc. |
| **Project Type** | Reforestation, Agroforestry, etc. |
| **Project Country** | Multi-select countries |
| **Methodology** | Filter by methodology |
| **Scorecard Range** | Filter by score thresholds |
| **Submission Date** | Date range filter |
| **Has Appeal** | Yes/No |
| **Override Status** | Has overrides / No overrides |

#### Applying Filters

1. Click the **filter icon** or filter bar
2. Select your filter criteria
3. Filters apply immediately
4. Clear all with the **'Clear'** button

#### Saved Filters

Save frequently used filter combinations:
1. Apply your filters
2. Click **'Save Filter'**
3. Name the saved view
4. Access from the filter dropdown

---

### Proposal Detail Page

#### Accessing Details

Click on any proposal row to open the detail page.

#### Page Sections

| Section | Content |
|---------|---------|
| **Summary** | Key project data, status, scorecard overview |
| **Project Information** | Full questionnaire responses |
| **Documents** | Uploaded files (PDD, KML, etc.) |
| **Pricing** | Volume and price offers |
| **Scorecard** | Full score breakdown |
| **Notes** | Internal team notes |
| **History** | Submission and stage history |

#### Header Information

- Submission timestamp
- Current stage indicator
- Proposal status
- Quick actions

#### Navigation

- Use **tabs** to navigate between sections
- **Back** button returns to table
- **Previous/Next** arrows to move between proposals

#### Quick Actions

From the proposal detail page:
- **Advance Stage** - Move to next stage
- **Reject** - Reject proposal (requires reason)
- **Add Note** - Add internal note
- **Add to Shortlist** - Add to a custom view
- **Export** - Export proposal data

---

### Pipeline View

#### Stage Pipeline

View proposals as a pipeline:
- Visual representation of each stage
- Count of proposals at each stage
- Click to filter to that stage

#### Pipeline Stages

```
Eligibility Check -> Screening Phase 1 -> Screening Phase 2 ->
Intermediate DD -> Member Opt-in -> Full DD
```

---

### Understanding Status

#### Proposal Status Types

| Status | Meaning |
|--------|---------|
| **Draft** | Supplier hasn't submitted yet |
| **Submitted** | Awaiting review |
| **Under Review** | Being evaluated |
| **Passed** | Advanced to next stage |
| **Rejected** | Did not pass stage |
| **Appeal Pending** | Supplier appealing rejection |

#### Status Indicators

- **Color badges** show status at a glance
- **Icons** indicate special conditions (appeal, new)
- **Timestamps** show last activity
